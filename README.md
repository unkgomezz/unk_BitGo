# BitFinderGo v0.5

# Instruções para rodar o código

## Requisitos
  -  Go instalado;
  -  Terminal CMD;

## Execução no CMD
Tutorial de execução:

 * No CMD, clone o repositório:
  ``` git clone git@github.com:unkgomezz/unk_BitGo.git```
 * Entre na pasta do projeto:
  ``` cd NomeDaPasta ```
 * Instale as dependências:
 ``` go mod tidy ```
 * Execute o comando:
 ``` go build -o btcgo.exe ./src ``` 
 * Execute o comando:
 ``` btcgo ```
